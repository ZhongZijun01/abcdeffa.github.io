GMOJ S5257 【小X的佛光】

Description

小 X 作为一名远近闻名、热爱数学的学佛、男孩子。他给出了你一个有 $n$ 个点的树，以及 $m$ 个询问，每次询问你从点 $a_i$ 到点 $b_i$ 的简单路径和从点 $c_i$ 到点 $b_i$ 的简单路径有多少个公共点。

Solution

猜了个结论：答案为这三个点中 LCA 深度最大的那个 LCA 到 $b_i$ 的距离。

简单地把各种情况都试了一下发现没有问题就用了。

因为 OJ 的系统栈比较小，所以遍历树的时候要用 BFS。

时间复杂度 $O(m \log n)$。
